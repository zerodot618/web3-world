# WEB3-WORLD

## Dictory struct

```
|--web3-world
|----next-app // frontend app
|----README.md
```

## How run

### next-app

Into next-app dictory and run:

```
npm install
npm run dev
```

## How deploy

### next-app

You can deploy next-app on [Vercel](https://vercel.com/).
